# Shared Types

Espacio reservado para tipos TypeScript compartidos, adaptadores o tipos generados a partir de los JSON Schemas.

## Uso previsto

- contratos consumidos por `apps/web` y `services/api`;
- mapping de enums y DTOs comunes;
- futuras utilidades de serializacion y validacion.

En esta etapa no se agregan tipos concretos para evitar duplicar contratos antes de definir la estrategia de generacion o sincronizacion.
